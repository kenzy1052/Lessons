Algebraic Expressions - Teacher Notes

Introduction to Algebraic Expressions

Algebraic expressions are mathematical phrases that can contain ordinary numbers, variables (like x or y), and operators (like add, subtract, multiply, and divide). They help us represent real-world situations in a mathematical way.

Teaching Tip

Start with concrete examples that students can relate to before moving to abstract algebraic concepts. Use visual aids and manipulatives to help students understand the relationship between words and algebraic expressions.

B7.2.2.1.1: Creating Simple Algebraic Expressions

From Mathematical Statements

Help students translate words into algebraic expressions. If x represents an unknown number:

Examples:
10 more than a number x → x + 10
5 less than a certain number x → x - 5
3 times a number x → 3x
Half of a certain number x → ½x or x/2
2 more than 5 times a certain number x → 5x + 2
When 8 times a certain number x is subtracted from 5 and the result is multiplied by 2 → 2(5 - 8x)
Student Tip

Look for keywords that indicate operations: "more than" or "sum" for addition (+), "less than" or "difference" for subtraction (-), "times" or "product" for multiplication (×), and "half" or "divided by" for division (÷).

From Real-Life Situations

Connect algebra to real-world problems that students can understand:

Examples:
Afrako is 3 years older than Maako. If Maako is now x years old, what is Afrako's age? → x + 3
Agbolosu and Tetteh were given GH₵400.00 to share. Tetteh had GH₵35.00 more than Agbolosu. If Agbolosu's share is x, write an expression for Tetteh's share. → x + 35
Find the profit a woman makes if she buys a basket of oranges for x cedis and sells it for y cedis. → y - x
Find the area of a rectangle which is t metres long and q metres wide. → t × q or tq
Find the perimeter of a rectangle which is x metres long and y metres wide. → 2(x + y)
Try It Yourself!

Create an algebraic expression for this situation: "A book costs b cedis. A pen costs 5 cedis less than the book."

B7.2.2.1.2: Addition and Subtraction of Algebraic Expressions

Adding Algebraic Expressions

When adding algebraic expressions, we combine like terms (terms with the same variable and exponent).

Examples:
Write each of these expressions in their simplest form:
  • x + x = 2x
  • y + y + y + y = 4y
  • s + s + s + t + t + k + k + k = 3s + 2t + 3k
Simplify the following expressions:
  • 4x + 3x + x = 8x
  • 5x + 4x + 2x + 3x = 14x
  • 3abc + 4abc + 2abc = 9abc
Student Tip

Think of variables as objects: if you have 4 apples and then get 3 more apples and then 1 more apple, you have 8 apples total. Similarly, 4x + 3x + x = 8x.

Subtracting Algebraic Expressions

When subtracting algebraic expressions, we subtract the coefficients of like terms.

Examples:
Write each of these expressions in their simplest form:
  • 5x - 2x = 3x
  • 3x - 4x - 2x = -3x
  • 7x - 4x - x = 2x

Adding and Subtracting Mixed Expressions

When expressions have different variables or constants, group and combine like terms.

Examples:
Simplify the following expressions:
  • 5x + 4 - 9y + 3x + 2y - 7 = (5x + 3x) + (-9y + 2y) + (4 - 7) = 8x - 7y - 3
  • 2p - 3q + 3p + 5q = (2p + 3p) + (-3q + 5q) = 5p + 2q
  • 4x + 7 - 2x - 4 = (4x - 2x) + (7 - 4) = 2x + 3
  • 7xy + 5x - 4x + 2xy - 3 = (7xy + 2xy) + (5x - 4x) - 3 = 9xy + x - 3
Practice Problem

Simplify: 3a + 4b - 2a + 5b - 1

B7.2.2.1.3: Multiplication and Division of Algebraic Expressions

Multiplying Algebraic Expressions

When multiplying algebraic expressions:

  • Multiply the coefficients (numbers)
  • Multiply the variables by adding their exponents
Examples:
Simplify the following expressions:
  • 4p × 8p² = 4 × 8 × p¹ × p² = 32p³
  • 5xy² × 4x⁴y³ = 5 × 4 × x¹ × x⁴ × y² × y³ = 20x⁵y⁵
  • -2a × 4c × 5b = -2 × 4 × 5 × a × c × b = -40abc
  • -3xy × 5y = -3 × 5 × x × y × y = -15xy²
Teaching Tip

Use visual models to show multiplication of variables. For example, show x × y as a rectangle with side lengths x and y, and x² as a square with side length x.

Dividing Algebraic Expressions

When dividing algebraic expressions:

  • Divide the coefficients (numbers)
  • Divide the variables by subtracting their exponents
Examples:
Simplify the following expressions:
  • 12x³y² / 16xy⁴ = (12/16) × (x³/x¹) × (y²/y⁴) = (3/4) × x² × y⁻² = 3x²/4y²
  • -30a²b³c / 6ab³c⁻² = (-30/6) × (a²/a¹) × (b³/b³) × (c¹/c⁻²) = -5 × a¹ × b⁰ × c³ = -5ac³
  • 18x⁵y² / 24x⁷y² = (18/24) × (x⁵/x⁷) × (y²/y²) = (3/4) × x⁻² × y⁰ = 3/4x²
Student Tip

Remember that any number to the power of 0 equals 1 (like y⁰ = 1), and negative exponents mean the variable is in the denominator (like x⁻² = 1/x²).

B7.2.2.1.4: Substituting Values to Evaluate Algebraic Expressions

To evaluate an algebraic expression, substitute the given values for the variables and perform the calculations.

Example 1:

Simplify the following expressions and substitute the values to evaluate them, if x = 2, y = 4, p = 3 and z = -1.

  • 3xy × 5y = 15xy² = 15(2)(4)² = 15(2)(16) = 480
  • 7xy + 5x - 4x + 2xy - 3 = 9xy + x - 3 = 9(2)(4) + 2 - 3 = 72 + 2 - 3 = 71
  • 4p × 8z² = 32pz² = 32(3)(-1)² = 32(3)(1) = 96
  • 5x + 4 - 9y + 3x + 2y - 7 = 8x - 7y - 3 = 8(2) - 7(4) - 3 = 16 - 28 - 3 = -15
Example 2:

Simplify the following expressions and substitute the values to evaluate them, if x = 2, y = 4, a = 3, b = 2, z = 1 and c = -1.

  • 12x³y² / 16xy⁴ = 3x²/4y² = 3(2)²/4(4)² = 3(4)/4(16) = 12/64 = 3/16
  • -30a²b³c / 6ab³c⁻² = -5ac³ = -5(3)(-1)³ = -5(3)(-1) = 15
  • 18x⁵y² / 24x³y² = 3x²/4 = 3(2)²/4 = 3(4)/4 = 12/4 = 3
  • 8xyz / 16xy = z/2 = 1/2
  • 5ab² / ab = 5b = 5(2) = 10
  • 21x⁷ / 3x⁴ = 7x³ = 7(2)³ = 7(8) = 56
Example 3:

If l = 5, a = 8, b = 3, h = 6, find the perimeter and area of the following shapes:

Shape Perimeter Formula Area Formula Perimeter Area
Rectangle (l=5, b=3) 2(l + b) l × b 2(5 + 3) = 16 5 × 3 = 15
Square (a=8) 4a a² 4(8) = 32 8² = 64
Triangle (b=3, h=6) Sum of sides ½ × b × h Depends on sides ½ × 3 × 6 = 9
Evaluate the Expression

Evaluate 3x² + 2y - 5 when x = 2 and y = 3

B7.2.2.1.5: Using Properties of Operations to Simplify Algebraic Expressions

We can use properties of operations (commutative, associative, distributive) to simplify algebraic expressions with rational coefficients.

Examples:
3xy × 2 + 6x²y³ / 2y² = 6xy + 3x²y
(7x + 4x - 2x) / 3x = 9x / 3x = 3
3x²y + 2xy² - 4x²y - 6xy² = (3x²y - 4x²y) + (2xy² - 6xy²) = -x²y - 4xy²
(15p³q² × 12x⁵y³) ÷ (36pq × 45x²y) = (180p³q²x⁵y³) ÷ (1620pqx²y) = (180/1620) × p²q¹x³y² = (1/9)p²qx³y²
(7x² + 2x²) / 3x² = 9x² / 3x² = 3
7a - 7a³ + 14a⁴ = 7a(1 - a² + 2a³)
Teaching Tip

Emphasize the importance of following the order of operations (PEMDAS/BODMAS): Parentheses/Brackets, Exponents/Orders, Multiplication and Division (from left to right), Addition and Subtraction (from left to right).

Student Tip

Look for common factors that can be factored out to simplify expressions. For example, in 7a - 7a³ + 14a⁴, 7a is a common factor in all terms.

Teaching Strategies

Visual Learning

Use visual representations to help students understand abstract concepts:

  • Use algebra tiles to represent expressions and operations
  • Draw diagrams to show real-world applications
  • Use color coding to distinguish between different variables and terms

Progressive Difficulty

Start with simple expressions and gradually increase complexity:

  • Begin with single-variable expressions
  • Progress to multiple variables
  • Introduce exponents after students master basic operations

Real-World Connections

Help students see the relevance of algebra in everyday life:

  • Use examples from shopping (discounts, taxes)
  • Connect to sports statistics
  • Relate to measurement and geometry problems

Collaborative Learning

Encourage peer-to-peer learning:

  • Pair students to check each other's work
  • Use group problem-solving activities
  • Have students create their own word problems

Assessment Ideas

Formative Assessment

  • Exit tickets with 1-2 problems to check understanding
  • Quick quizzes on specific skills
  • Observation during group work
  • Student self-assessment of confidence with concepts

Summative Assessment

  • Unit tests covering all concepts
  • Project where students create a guide to algebraic expressions
  • Real-world problem-solving tasks
  • Peer teaching presentations
