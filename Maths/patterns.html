Understanding Patterns and Relations

What are Patterns and Relations?

Key Definitions
Pattern: A set of numbers, shapes, or objects that follow a certain rule.
Relation: The way two sets of numbers (input and output) are connected.
Example:

If we have the relation x → 2x, then:

When x = 1 → output = 2 × 1 = 2
When x = 2 → output = 2 × 2 = 4
When x = 3 → output = 2 × 3 = 6

So the relation is a pattern that doubles the input number.

Teaching Tip

Start with concrete examples that students can relate to before moving to abstract concepts. Use visual aids and manipulatives to help students understand the relationship between inputs and outputs.

Try It Yourself!

If the relation is x → 3x + 1, what is the output when x = 4?

Extending Patterns

Matchstick Pattern Example

Pattern:
1
5 sticks
2
8 sticks
3
11 sticks
4
14 sticks
5
17 sticks

Each new pattern adds 3 more sticks than the previous one.

Rule: Add 3 each time.

Number Pattern Example

Pattern:
Domain (Input) 1 2 3 4 5
Co-domain (Output) 5 7 9 11 13

Each output increases by 2.

Rule: x → 2x + 3

Check:

For x = 1 → (2×1) + 3 = 5
For x = 2 → (2×2) + 3 = 7 ✅

Describing Rules Using Words

Converting Words to Expressions

We can describe relations in words:

Examples:
One more than twice the number → x → 2x + 1
Three less than five times the number → x → 5x – 3
Half the number plus 4 → x → (x ÷ 2) + 4
Example:

If input is 4, and rule is "twice the number plus 1":

Output = (2×4) + 1 = 9

Student Tip

Look for keywords that indicate operations: "more than" or "sum" for addition (+), "less than" or "difference" for subtraction (-), "times" or "product" for multiplication (×), and "half" or "divided by" for division (÷).

Using Patterns to Solve Problems

Breakfast Costs Example

Problem:
Number of workers 1 2 3 4 5 6 ?
Cost (GH₵) 3 6 9 12 15 18 120

Rule: x → 3x

If total cost is 120, then:

120 ÷ 3 = 40 workers.

Predicting Values Example

Problem:
Domain (x) 1 2 3 4 5 9
Rule: x → 3x – 1 2 5 8 11 14 ?

For x = 9 → (3×9) – 1 = 26

Graphing Relations

Plotting Relations on a Graph

We can show relations on a number plane (graph).

Example Rule: x → 2x + 1
Domain -2 -1 0 1 2 3
Output -3 -1 1 3 5 7

Plot these points: (-2, -3), (-1, -1), (0, 1), (1, 3), (2, 5), (3, 7).

They form a straight line.

Practice Questions and Summary

Quick Summary

Key Points to Remember
A pattern follows a rule.
A relation connects input (x) to output (y).
Rules can be written in words or symbols.
Example: x → 3x + 2.
We can use rules to extend patterns.
We can use rules to make predictions.
We can use rules to solve word problems.
We can draw graphs to visualize relations.
Graphs help us visualize relations on the number plane.
